# Getting started

A simple rails API

Clone this respository and run the commands:

OBS: configure your database.yml if you need

```
bundle install
rails db:create db:migrate dev:setup
bundle exec rspec
```

## Built with

- Rails 5.2.0 API mode
- Rpec with shoulda matchers for tests
- MySQL
